: 【新唐人2020年08月22日訊】近日，印度古吉拉特邦（Gujarat）發生驚魂一幕。一名年
: 輕男子睡午覺時感覺被重物壓住，醒來時看到一隻母獅子壓在胸口狠狠瞪着他，最後男子
: 使出全力將母獅推開，成功地救了自己一命。
: 據《印度時報》8月17日報導，事件發生在古吉拉特邦Amreli區的Abharampara村。當時維
: 普·克萊亞在他的小屋裡睡著了，他突然感到有「龐然大物」壓在胸口上，讓他難以呼吸
: 。他睜開眼睛，竟看到一頭母獅子正坐在他的胸口，狠狠瞪著他。
: 但克萊亞盡量保持理智，用盡全身力氣將這頭獅子推開。這頭母獅似乎被凱萊亞的舉動嚇
: 到，不但沒有襲擊，還落荒而逃，走進草叢和樹林裡。
: 克萊亞事後回憶說，這頭母獅一定已經意識到他不是獵物，覺得選擇攻擊他是一個巨大的
: 錯誤。他說，當時他正在自己的小屋裡睡覺，突然獅子的一聲吼叫把他嚇醒了，獅子沉重
: 的身軀壓在他的胸口。
: 克萊亞說，他當時凝視著一雙仿佛能看到死亡的眼睛。「謝天謝地，我很快就察覺到發生
: 了什麼，本能地控制住自己的理智，聚集了我所有的力量，把這頭母獅從我身邊推了下來
: 。」
: 該村村長克曼（Bhupendra Khuman）說：近期有一些獅群經常在賈巴爾村與阿巴拉漢帕拉
: （Abhrampara）地區出現，導致類似事件時常發生，許多民眾都能看到野生的獅子。
: 2月份，5歲小男孩基紹爾·帕爾馬，在離家數英里以外的地方獨自睡覺時，被一頭母獅發
: 現，不幸被獅子咬死。科曼認為，最近可能還會發生這種人與動物的衝突。
: （記者李芸報導/責任編輯：李泉）
